Importance of Coding Standard

Benefits of Following Coding standards:

1) The code is modular

2) Changes or future enhancement does not takes up lot of time.

3) Coding standards greatly reduces development time and development cost.

4) Resources can be easily switched between the projects and can follow the code.

5) Minimizes Performance Pitfalls.

6) Maintenance of project becomes easy.

Common Aspects and rules of Coding Standards:

1) Naming Conventions

2) Organization of files

3) Formatting and Indentation

4) Comments and Documentation

5) Modularity and Reusability of code

6) Portability

7) Designs
